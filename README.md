# wow-discord-rank-sync
Automates Discord role assignment based on WoW guild ranks using GRM data.

# Python Project Setup
1. Clone the repository
2. Copy `.env.example` to `.env`
3. Enter the required environment variables in `.env`
    - `DISCORD_TOKEN` is the Bot Token generated in the Discord Developer Portal
    - `TEST_USER_ID` is the Discord User ID manipulated by tests
4. Create a local virtual environment with `python -m venv .venv`
5. Activate the virtual environment with `source .venv/Scripts/activate` (in bash) or `.venv\Scripts\activate` (in cmd)
6. Install dependencies with `pip install -r requirements.txt`
